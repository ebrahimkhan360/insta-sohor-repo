2.what does JavaScript event loop do?

Event loop is a JavaScript runtime uses a message row, which is a list of processed messages. At some point during the event loop, runtime starts handling messages in rows, starting with the oldest. To do this, the message is moved from the row and its corresponding function is called with the message as an input parameter. As always, calling a function creates a new stack frame for the use of that function.
